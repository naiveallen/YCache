
1. Import project to IDEA or Eclipse.
2. In the IDEA, import the maven dependencies automatically.
3. Edit Config.properties, add or edit cluster host address.
4. Configure 5 Main application startup, input program argument "-port {port}", then start them one by one.
5. Configure client application startup, input program argument "-host {host_addr}", start it.
6. In client, use GET or PUT command to test key/value cache.
